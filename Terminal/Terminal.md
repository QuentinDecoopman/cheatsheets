# Commandes de terminal

- [Index](/Readme.md)

## Navigation dans le système de fichiers

### Lister les fichiers et répertoires

```sh
ls
```

### Changer de répertoire

```sh
cd NomDuDossier
```

### Afficher le répertoire de travail courant

```sh
pwd
```

## Manipulation des fichiers et répertoires

### Créer un fichier

```sh
touch fichier.txt
```

### Créer un nouveau répertoire

```sh
mkdir NouveauDossier
```

### Copier des fichiers ou répertoires

```sh
cp fichier.txt destination/
```

### Déplacer des fichiers ou répertoires

```sh
mv fichier.txt destination/
```

### Supprimer des fichiers

```sh
rm fichier.txt
```

## Gestion des fichiers

### Afficher le contenu d'un fichier

```sh
cat fichier.txt
```

## Recherche de fichiers

### Rechercher des fichiers dans un répertoire

```sh
find . -name "*.txt"
```

## Gestion des processus

### Afficher les processus en cours

```sh
ps aux
```

## Informations système

### Afficher des informations sur le système

```sh
uname -a
```

### Afficher l'utilisation de la mémoire

```sh
free -h
```

### Afficher l'utilisation de l'espace disque

```sh
df -h
```
